# Referências úteis (sem datas antigas)

- RasQberry – projeto que mostra setup de Qiskit no Raspberry Pi e demos com Sense HAT. Guia e repositório oficiais.
  - Hackster resumo: https://www.hackster.io/news/jan-and-robert-lahmann-get-a-quantum-computer-running-on-your-raspberry-pi-in-under-30-minutes-4b972010009d
  - Repo RasQberry: https://github.com/JanLahmann/RasQberry
  - Demos RasQberry: https://rasqberry.org/quantum-computing-demos/demo-list
- Qrasp – projeto original com **Sense HAT** para visualizar circuitos quânticos usando **Qiskit** no Raspberry Pi.
  - Artigo: https://medium.com/qiskit/qrasp-a-wee-quantum-computer-74ef7f927b1e
  - Post Raspberry Pi: https://www.raspberrypi.com/news/ibm-q-system-one-quantum-computing-raspberry-pi/
- Instalação do Qiskit (1.x): https://quantum.cloud.ibm.com/docs/guides/install-qiskit
- Instalação do PennyLane: https://pennylane.ai/install
- Biblioteca Sense HAT (Python): https://github.com/astro-pi/python-sense-hat

> Observação: este repositório usa simuladores clássicos no Raspberry Pi e pode opcionalmente conectar a backends remotos (via contas dos provedores).
